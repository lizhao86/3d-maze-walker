# 3D Maze Walker

A first-person cyberpunk maze prototype built with Vite, TypeScript, and Three.js.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Controls

- `WASD`: move
- Mouse: look around
- `E`: open chests or use the exit
- Left click: attack
- `1-7`: switch weapons

## Current Prototype

- Random maze generation
- First-person movement and wall collision
- Cyberpunk wall and floor materials
- Chests with weapons, ammo, med kits, and key fragments
- Skull and zombie monsters
- HUD with HP, objective, minimap, hotbar, ammo, and crosshair
